## Project Summary

A lightweight web application deployed across AWS, Azure, and GCP that monitors deployment health, displays infrastructure status, records health check history, and demonstrates multi-cloud failover capability.

The application serves as the workload for a Terraform-based multi-cloud deployment system demonstrating:
- Automated multi-cloud deployment
- Load balancing and failover
- Disaster recovery
- Infrastructure monitoring
